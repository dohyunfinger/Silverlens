# SilverLens 제품 소개서

AI가 더 똑똑해지는 것보다, 사용자를 먼저 이해해야 진짜 기술이라고 생각합니다.
SilverLens는 기술이 발전할수록 소외되는 어르신들을 위해, 고등학생의 시선과 기술로 만든 시니어 맞춤형 AI 조수 프로젝트입니다.


### 서비스 개요

시니어가 기계에 맞추는 것이 아닌, AI가 어르신을 먼저 이해하는 '눈과 귀' 서비스를 지향합니다. 디지털 소외를 해소하고, 차가운 기술 속에서 따뜻한 정서적 유대감을 전하는 인간 중심 기술을 목표로 합니다.


### 우리가 발견한 진짜 문제들

저희 팀은 처음에 '보이스피싱 예방 시스템'을 기획했습니다. 하지만 개발을 준비하면서 "이걸 우리 할머니가 혼자 쓰실 수 있을까?" 하는 의문이 들었습니다. 노트북을 덮고 일주일 동안 가족들의 스마트폰 생활을 관찰하며 책상 위에서는 몰랐던 진짜 문제들을 발견했습니다.

손끝이 둔해진 어르신들에게는 작은 스마트폰 자판으로 글자를 하나하나 입력하는 것 자체가 거대한 숙제였습니다. 자판 대신 음성 인식을 켜도, 기존 AI들은 어르신들의 깊은 사투리나 시장에서 쓰는 익숙한 단어(정구지, 정종 등)를 전혀 알아듣지 못하고 오류만 냈습니다.

게다가 마트나 미디어에 쏟아지는 '로즈마리'나 '아보카도' 같은 외국 식재료 이름을 잘 모르셔서 답답해하셨고, 인터넷에서 몸에 좋다는 건강 정보를 보시면 무조건 많이 먹을수록 좋은 거라고 굳게 믿으시는 위험한 상황도 보았습니다.

> **[기존 AI들은 사용자가 표준어를 정확히 타이핑하고 기계의 방식대로 질문하기만을 기다리고 있었습니다. 어르신들이 기술을 못 쓰는 게 아니라, AI가 어르신들의 삶을 배려하지 않고 있었습니다.]**


### 우리의 해결책 (핵심 기능)

1. 사투리 찰떡 음성 인식
기존 AI는 표준어로 정확하게 발음하지 않으면 질문 자체를 이해하지 못하고 오류가 발생합니다. SilverLens는 사투리와 생활 용어 데이터를 매핑하여, 어르신이 "이 정구지 어째 먹노?"라고 투박하게 말씀하셔도 "부추 요리법"으로 정확히 알아듣고 손주처럼 답변합니다.

2. 돋보기 카메라
기존 AI는 단순히 사진 속 사물의 이름만 딱딱한 텍스트로 보여주고 끝납니다. SilverLens는 카메라로 식재료를 비추면 쉬운 우리말로 번역하고, 공공 영양 데이터와 연동하여 "할머니, 이 음식은 당뇨에 안 좋으니 조금만 드세요!" 하며 건강 맞춤형 안전 경고를 전달합니다.

3. 손주 감성 UI/UX
복잡한 카테고리와 작은 아이콘이 많으면 어르신들이 오조작하거나 기계 자체를 두려워하게 됩니다. SilverLens는 복잡한 화면을 다 지우고 [말하기]와 [사진 찍기] 두 개의 초대형 버튼으로만 화면을 구성했습니다. 여기에 실제 손주의 목소리 톤을 학습한 TTS를 적용해 친근감을 극대화했습니다.


### 우리 팀의 다짐

> **기술이 고도화될수록 소외받는 어르신들이 생겨난다면, 고등학생인 저희가 배운 기술로 그 격차를 따뜻하게 메워보고 싶습니다.**

대한민국 모든 어르신들이 자식이나 타인의 도움 없이도 당당하고 안전하게 디지털 세상을 누릴 수 있도록, 저희 SilverLens 팀은 이 발걸음을 멈추지 않겠습니다.
