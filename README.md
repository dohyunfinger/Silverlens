# SilverLens 제품 소개서

### 서비스 개요
> **[SilverLens는 시니어가 기술에 맞추는 것이 아니라, 기술이 시니어를 이해하는 '눈과 귀'가 되어주는 AI 서비스입니다.]**

<br>

### 핵심 기능

#### 1. 사투리·시장 언어 음성 인식
* **문제점**: 어르신들은 타이핑이 어렵고, 사투리나 시장 표현을 쓰면 기존 AI가 알아듣지 못함.
* **해결책**: OpenAI Whisper API를 사투리 데이터로 학습시켜 "이 정구지 어째 먹노?"라고 해도 "부추 요리법을 찾으시나요?"라며 찰떡같이 이해하고 답합니다.

#### 2. 식재료·건강정보 '돋보기 카메라'
* **문제점**: 아보카도 같은 외국 식재료 이름을 잘 모르고, 인터넷 건강 정보를 과다 신뢰함.
* **해결책**: 카메라로 찍으면 쉬운 우리말로 번역해 주고, 공공 데이터를 기반으로 "할머니, 이 음식은 당뇨에 안 좋으니 조금만 드세요!" 하고 위험을 알려줍니다.

#### 3. 손주처럼 따뜻한 화면과 목소리
* **문제점**: 차가운 AI 화면보다 사람에게 물어보는 것이 더 편함.
* **해결책**: 복잡한 메뉴 대신 [말하기]와 [사진 찍기] 큰 버튼 2개로만 구성하고, 실제 손주의 목소리 같은 TTS를 적용해 친근함을 줍니다.

<br>

### 기대 효과
> **[저희는 AI가 더 똑똑해지는 것보다 사용자를 먼저 이해해야 진짜 기술이라고 생각합니다.]**

* **소외 없는 기술**: 타이핑을 못 해도, 사투리를 써도 누구나 쉽게 AI를 씁니다.
* **안전한 건강 정보**: 인터넷의 검증되지 않은 정보로부터 어르신을 지킵니다.
