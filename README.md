# 🔍 SilverLens (실버렌즈)
> **Silver + Lens = 기술을 배우기 힘든 시니어를 위해, 어르신의 말투를 먼저 배우는 AI 식생활 도우미**

고령층 노인들은 노안으로 인한 디지털 정보 소외, 시장 사투리 사용으로 인한 기술적 소외, 그리고 미디어 맹신으로 인한 잘못된 식습관 형성이라는 삼중고를 겪고 있습니다. **SilverLens**는 고령층 사용자가 평소 쓰시던 말투와 일상적인 행동(말하기, 사진 촬영)만으로 국책기관의 검증된 식재료 건강 정보를 완벽하게 제공받을 수 있도록 설계된 **시니어 맞춤형 하이브리드 AI 로컬라이징 서비스**입니다.

---

## 🚨 문제 정의 (Problem Definition)

### 1. ⌨️ 디지털 입력 장벽으로 인한 정보 소외
* **자판 입력의 한계** : 현재 제공되는 대부분의 건강 정보 서비스는 자판으로 정확한 표준어 키워드를 타이핑해야만 정보를 반환합니다. 손떨림이나 노안을 겪는 시니어들은 서비스 진입 단계부터 극심한 피로감을 느끼고 이탈합니다.
* **소통 격차의 발생** : 전통시장이나 일상에서 흔히 쓰이는 '정구지(부추)', '다마네기(양파)' 같은 구어체 방언이나 옛 명칭은 기존 검색 엔진에서 인식되지 않아 올바른 정보를 얻기 어렵습니다.

### 2. ⚠️ 미디어 맹신과 직관적 영양 가이드의 부재
* **무분별한 과다 섭취** : "인터넷이나 TV에서 특정 식재료가 몸에 좋다"고 하면, 그 부작용을 간과한 채 무작정 모든 음식에 과량 투하하여 섭취하는 잘못된 식습관이 형성됩니다.
* **난해한 정보 단위** : 복잡한 영양 성분표나 'g(그램)' 단위의 수치는 시니어가 일상에서 가늠하기 어렵습니다. 지병(당뇨, 고혈압 등)에 맞춘 직관적인 제한선과 일상 도구 기준의 계량 정보가 절실합니다.

---

## 💡 해결방안 및 핵심 가치 (Core Value)

* **제로 타이핑(Zero-Typing)** : 사용자가 자판을 두드릴 필요 없이 보기 쉬운 마이크 버튼을 누르고 말하거나, 생소한 외국 식재료/영어 상표명을 카메라로 찍기만 하면 AI가 식재료의 형태와 의도를 즉시 파악합니다.
* **찰떡같은 사투리 표준어 변환** : 전국의 전통시장 방언과 구어체 지식을 탑재한 LLM을 활용해, 무질서하게 쏟아지는 어르신의 사투리 원문을 정확한 표준어 데이터와 실시간으로 매핑합니다.
* **RAG 기반 신호등 안전 가이드** : AI가 가짜 정보를 지어내는 할루시네이션을 0%로 차단하기 위해 공신력 있는 국책기관 DB를 기반으로 작동합니다. 사용자의 지병 정보를 반영하여 유해 식재료일 경우 빨간색 경고등을 켜주고, "종이컵 반 컵", "한 숟가락" 등 시니어 눈높이에 맞춘 눈대중 계량 가이드를 화면과 음성으로 동시 제공합니다.

## 🛠️ 기술 아키텍처 (Architecture) 수정필요

### 🔄 데이터 및 서비스 흐름 (Data & Service Flow)

* **클라이언트 (프런트엔드)** : 앱 진입 후 선호 언어(한국어/영어/일본어) 및 지병(당뇨/고혈압 등) 입력 ➡️ 메인 화면에서 사투리 음성 입력 또는 식재료 사진 촬영 ➡️ 최종 화면에서 지병 데이터가 실시간 반영된 RAG 기반 맞춤형 가이드(큰 글씨 UI) 확인 및 음성 자동 안내 청취
* **서버 (백엔드)** : 클라이언트의 멀티모달 입력 데이터 수집 및 정제 ➡️ 국책기관 Open API 기반 지식 저장소(Vector DB) 연동 ➡️ 로컬 음성 처리 모델 및 클라우드 생성형 AI 오케스트레이션 제어 ➡️ 응답 텍스트 가공 및 오디오 스트리밍 파이프라인 관리
* **AI 파이프라인** :
  * **Faster-Whisper (On-Device/로컬 AI)** : 시장 소음이 섞인 어르신의 음성을 서버 대기 시간(Latency) 없이 기기 자체에서 초고속으로 텍스트(STT) 변환 처리
  * **Gemini API (Cloud/멀티모달 LLM)** : 입력된 [식재료 사진 + 사투리 텍스트 + 유저 지병 컨텍스트]를 단 1회의 호출로 결합. 방언 해석, 외국어 상표 번역, 시각적 사물 인지, 시니어 친화적 구어체(~하셔요) 텍스트 생성을 통합 수행
  * **Gemini 2.5 Flash TTS (나레이션 Engine)** : 생성된 맞춤 가이드 원문을 고품질의 친근한 손자같은 목소리로 실시간 변환 (네트워크 단절 및 API 초과 시 **Web Speech API 브라우저 TTS**가 즉시 백업 가동)
* **데이터베이스 (Supabase)** : 식품의약품안전처(영양 성분), 농촌진흥청(국가표준식품성분표), 질병관리청(만성질환 식사 가이드) 등 국책기관 오픈 데이터를 사전에 수집·정제하여 RAG 참조용 임베딩 데이터 및 사용자 맞춤 프리셋 관리
